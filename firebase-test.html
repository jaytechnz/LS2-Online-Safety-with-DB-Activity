Most Common Causes

  • Firestore Database has not been created.
  • The project has Realtime Database instead of Firestore Database.
  • firebase-config.js still contains placeholder values.
  • Firestore rules do not allow create and read.
  • The site is opened as a local file:/// page instead of from GitHub Pages, Firebase Hosting, Netlify, or another web host.
  • A school network filter is blocking Firebase.
