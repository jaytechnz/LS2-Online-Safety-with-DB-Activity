Firebase version 2

Fixes:
- Students now progress through Case Study 1, Case Study 2, and Case Study 3 using Next/Back buttons.
- The Submit button only appears on Case Study 3.
- summaries.html now shows clear Firebase error messages instead of staying on "Loading group responses..." forever.

Setup:
1. Paste your Firebase config into firebase-config.js.
2. Create a Firestore Database.
3. Use the rules in firestore-rules.txt for a short controlled classroom activity.
4. Upload all files to your repo/host.
5. Open the hosted index.html page, not the local file version.
